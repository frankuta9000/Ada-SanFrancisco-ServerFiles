## WARNING
You cannot monetize any server that is running Ada Alpha! This includes even things such as selling priority queue slots.

## Regenerating mapgroupproto
If you plan to `exportproxydata` and regenerate `mapgroupproto`, you have to manually add the helicopter crash entries listed in [`helicopters.txt`](helicopters.txt).

There is also a crane in Lovelace that spawns floating loot — you will have to delete that too.
